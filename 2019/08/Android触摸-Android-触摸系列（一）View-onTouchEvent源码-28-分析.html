一、前置知识16进制书写二进制标识

View源码中很多16进制标识int如下：

16进制对应2进制（用于做标记）
0x001 -> 0000 0001 | 0x010 -> 0001 0000
0x002 -> 0000 0010 | 0x020 -> 0010 0000
0x004 -> 0000 0100 | 0x040 -> 0100 0000
0x008 -> 0000 1000 | 0x080 -> 1000 0000
通过这组数据规律不难看出，16进制书写二进制01指定位置标识的方便之处

  • flag中添加标识

  • flag中移除标识
